## 1. 需要安裝的軟體

| 軟體 | 用途 | 備註 |

## 2. Windows 安裝步驟

### 2-1. 安裝 Git

1. 前往：https://git-scm.com/download/win
2. 下載 Windows 版安裝程式（`.exe`）
3. 執行安裝，所有選項保持預設，一路點「Next」→「Install」
4. 安裝完成後，在桌面空白處**按右鍵**，看到「**Git Bash Here**」即成功

### 2-2. 安裝 Python 3.11

1. 前往：https://www.python.org/downloads/
2. 點「**Download Python 3.11.x**」（選最新的 3.11 版）
3. 執行安裝程式
4. ⚠️ **安裝畫面最下方，務必勾選「Add Python to PATH」**（非常重要，漏勾會導致後續指令失敗）
5. 點「Install Now」等待完成
6. 驗證安裝：桌面右鍵 → Git Bash Here，輸入：
   ```bash
   python --version
   ```
   應顯示 `Python 3.11.x`

### 2-3. 安裝 VS Code

1. 前往：https://code.visualstudio.com/
2. 點「**Download for Windows**」
3. 執行安裝，安裝過程中勾選以下兩項（重要）：
   - **「Add "Open with Code" action to Windows Explorer file context menu」**
   - **「Add to PATH」**
4. 完成安裝

## 3. Mac 安裝步驟

### 3-1. 確認 Git（通常已內建）

1. 按 `Command + 空白鍵` → 輸入 `Terminal` → 按 Enter，開啟終端機
2. 輸入：
   ```bash
   git --version
   ```
3. 若顯示版本號碼（如 `git version 2.x.x`）即已安裝，可跳過
4. 若跳出安裝提示視窗，點「安裝」即可

### 3-2. 安裝 Python 3.11

1. 前往：https://www.python.org/downloads/
2. 點「**Download Python 3.11.x**」→ 選「**macOS 64-bit universal2 installer**」
3. 執行 `.pkg` 安裝程式，一路點「Continue」→「Install」
4. 驗證安裝：開啟 Terminal，輸入：
   ```bash
   python3 --version
   ```
   應顯示 `Python 3.11.x`

### 3-3. 安裝 VS Code

1. 前往：https://code.visualstudio.com/
2. 點「**Download for Mac**」（下載 `.zip`）
3. 解壓縮後，將「**Visual Studio Code**」拖入「**應用程式（Applications）**」資料夾
4. 開啟 VS Code
5. 按 `Command + Shift + P` → 輸入 `shell command` → 選「**Shell Command: Install 'code' command in PATH**」

## 4. 下載程式碼（Clone）

### Windows

桌面空白處右鍵 → 「**Git Bash Here**」，輸入：

```bash
cd ~/Desktop
git clone https://github.com/你的帳號/penghu-linebot.git
cd penghu-linebot
```

### Mac

開啟 Terminal，輸入：

```bash
cd ~/Desktop
git clone https://github.com/你的帳號/penghu-linebot.git
cd penghu-linebot
```

> 下載完成後，桌面會出現 `penghu-linebot` 資料夾，即為完整專案。

## 5. 安裝套件

### Windows

在 Git Bash 中（確認目前在 `penghu-linebot` 資料夾內）：

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### Mac

在 Terminal 中：

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

> ⏳ `pip install` 約需 **10–20 分鐘**，需下載 PyTorch 等大型套件，請耐心等候。
>
> 安裝成功後，終端最後一行應出現 `Successfully installed ...`。

**注意：每次重新開啟終端，都需要重新啟動虛擬環境：**

| 系統 | 啟動虛擬環境指令 |
|------|----------------|
| Windows | `venv\Scripts\activate` |
| Mac | `source venv/bin/activate` |

啟動後，終端左側會出現 `(venv)` 字樣，代表虛擬環境已啟用。

## 6. 設定環境變數（.env）

程式需要一個 `.env` 檔案存放所有 API 金鑰，此檔案不會隨程式碼下載，需手動建立。

1. 用 VS Code 開啟專案資料夾
2. 在根目錄新增一個名為 `.env` 的檔案
3. 複製以下內容貼入，並填入對應的值：

```env
# LINE Bot
LINE_CHANNEL_SECRET=填入你的Channel_Secret
LINE_CHANNEL_ACCESS_TOKEN=填入你的Channel_Access_Token

# LIFF
LIFF_ID=填入你的LIFF_ID

# AI（Mistral）
MISTRAL_API_KEY=填入你的Mistral_API_Key

# HuggingFace（Embedding 模型）
HUGGINGFACE_HUB_TOKEN=填入你的HuggingFace_Token

# 氣象署
OPENDATA_CWA_API_KEY=填入你的氣象署授權碼

# Email 通知
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=填入你的Gmail帳號
SMTP_PASS=填入Gmail應用程式密碼
NOTIFY_EMAIL=填入收件人信箱
```

4. 存檔（`Ctrl+S` 或 `Command+S`）

> 各 API 金鑰的申請方式請參考《交付文件》第 3 章。

## 7. 啟動程式

確認虛擬環境已啟動（左側有 `(venv)`），輸入：

### Windows
```bash
python app.py
```

### Mac
```bash
python3 app.py
```

看到以下訊息代表啟動成功：
```
* Running on http://127.0.0.1:5000
```

> 若看到錯誤訊息，最常見原因是 `.env` 檔案內有金鑰未填或填錯。

## 8. 本機測試（ngrok）

程式啟動後只能在本機執行，LINE 無法連線。需要 ngrok 建立對外通道。

### 8-1. 安裝 ngrok

1. 前往：https://ngrok.com/ 免費註冊
2. 下載對應系統版本

   - **Windows**：下載 `.zip`，解壓縮後把 `ngrok.exe` 放到專案資料夾
   - **Mac**：下載後解壓縮，或使用 Homebrew：
     ```bash
     brew install ngrok
     ```

### 8-2. 啟動 ngrok

**另開一個新的終端**（不要關掉跑 `python app.py` 的那個），輸入：

```bash
ngrok http 5000
```

畫面出現類似以下內容：
```
Forwarding  https://xxxx.ngrok-free.app -> http://localhost:5000
```

複製 `https://xxxx.ngrok-free.app` 這段網址。

### 8-3. 更新 LINE 設定

將複製的網址填入以下兩個地方：

| 位置 | 填入值 |
|------|--------|
| LINE Developers → Messaging API → Webhook URL | `https://xxxx.ngrok-free.app/callback` |
| LINE Developers → LINE Login Channel → LIFF → Endpoint URL | `https://xxxx.ngrok-free.app/liff/flight` |

填完後點「**Verify**」，出現 Success 即可用手機測試。

> ⚠️ 每次重新啟動 ngrok，網址都會改變，需重新填入上述兩個地方。

## 9. 常見問題

### `python` 或 `pip` 找不到（Windows）

Python 安裝時未勾選「Add Python to PATH」，請重新安裝並勾選。

### `venv\Scripts\activate` 出現權限錯誤（Windows）

開啟 PowerShell（以系統管理員執行），輸入：
```
Set-ExecutionPolicy RemoteSigned
```
之後改用 Git Bash 執行 `venv\Scripts\activate`。

### `pip install` 中途失敗

網路問題，重跑一次即可：
```bash
pip install -r requirements.txt
```

### 啟動後看到 `KeyError` 或 `ValueError`

`.env` 檔案有金鑰未填入，對照第 6 節逐一確認。

### ngrok 顯示 `ERR_NGROK_108`

ngrok 免費帳號同時只能開一個 session，關掉其他開著的 ngrok 再重試。

### LINE Webhook Verify 顯示 Timeout

伺服器尚未啟動完成，等待 30 秒後再點 Verify。
